# Code of Conduct

## Our standard

Be respectful, constructive, patient, and welcoming. Discuss ideas and behavior without attacking people. Harassment, discrimination, threats, doxxing, sexualized conduct, and deliberate disruption are not acceptable.

## Scope

This standard applies to repository issues, pull requests, discussions, security reports, and other project spaces.

## Enforcement

Report unacceptable behavior privately through the repository owner's GitHub profile or a private security advisory when the report is security-sensitive. Maintainers may edit or remove content, close discussions, reject contributions, or restrict participation when necessary to protect the community.

Enforcement decisions should consider context, impact, repeated behavior, and the safety of affected participants.
