# Support

## Questions and installation help

Use [GitHub Discussions](https://github.com/wilsongpt1/codex-danger-gate/discussions) for setup questions, usage guidance, and ideas that are not yet actionable bug reports.

## Bugs and feature requests

Use the repository's structured [issue forms](https://github.com/wilsongpt1/codex-danger-gate/issues/new/choose). Search existing issues before opening a new one.

## Security reports

Do not report security vulnerabilities in public issues or discussions. Follow [SECURITY.md](SECURITY.md) and use private vulnerability reporting.

## Scope

Community support is provided on a best-effort basis for the latest release on supported Windows and PowerShell environments. General Codex account, billing, entitlement, or product availability issues should be directed to OpenAI support.
